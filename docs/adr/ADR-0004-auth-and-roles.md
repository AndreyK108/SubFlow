# ADR-0004: Аутентификация и роли (Admin/User)

Дата: 2026-01-18
Статус: Proposed

## Контекст
Система B2C + упрощённый B2B: роли Admin и User.
Нужно разграничение доступа и простая реализация.

## Решение
- ASP.NET Core Identity (cookie-based).
- Роли: Admin, User.
- Авторизация по ролям на уровне страниц/эндпоинтов.

## Обоснование
- Identity — стандартная реализация, хорошо документируется и поддерживается.
- Роли покрывают требование упрощённого B2B.

## Последствия
- Таблицы Identity создаются миграциями EF Core.
- Для B2B-режима позже добавим OrganizationId и правила принадлежности данных.
